# 洛天依 HTTP 状态码表情
本项目受到 [HTTP Cat](https://http.cat/) 启发，制作了洛天依版的 HTTP 状态码表情包。

## 制作规格
162x228 像素，PNG 格式，包含透明 Aplha 通道（如有）。

## 已制作的状态码
 - `100 Continue`
 - `200 OK`
 - `201 Created`
 - `202 Accepted`
 - `204 No Content`
 - `206 Partial Content`
 - `301 Moved Permanently`
 - `302 Found`
 - `304 Not Modified`
 - `400 Bad Request`
 - `401 Unauthorized`
 - `402 Payment Required` （由阿绫代替）
 - `403 Forbidden`
 - `404 Not Found`
 - `405 Method Not Allowed`
 - `409 Conflict`
 - `410 Gone`
 - 此外，还有一个天依吃 Docker 的 GIF 表情代表 500 和其他情况。
